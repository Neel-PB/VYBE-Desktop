# Editor group watermark – Cursor reference

Use this file to paste **outerHTML** from Cursor’s editor area for both states: (1) no folder open, (2) project/workspace open but empty editor group. Replicate layout/copy for Vybe **without** Cursor logo/branding.

## Where to get the HTML in Cursor

1. Open Cursor with **no folder** (empty window).
2. In the main editor area you should see the watermark (letterpress + shortcuts).
3. Right‑click the watermark area → **Inspect** (or open DevTools).
4. In the Elements panel, find the watermark container (e.g. `.editor-group-watermark` or the parent of the centered content).
5. Right‑click that node → **Copy** → **Copy outerHTML**.
6. Paste below.
