# VYBE Settings – Sidebar & editor container comparison

Full editor container outerHTML for Cursor vs VYBE. Use this to align layout, spacing, and scrollbar behavior.

**Layout intent (Cursor):** `| sidebar | tab |` — total space (blank on both sides) expands with the window; only the **content** (sidebar content and tab content) has standard max widths. The scrollbar is on the right wall of the whole editor.

---

## 1. Theirs (Cursor) – full editor container

One scrollable wraps the **entire** layout (`cursor-settings-layout-main` = sidebar + pane-content). Vertical scrollbar is at `right: 0` of that scrollable = right wall of the editor. Sidebar and tab are flex siblings; container grows, content stays within max widths.
